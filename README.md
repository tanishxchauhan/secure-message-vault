# 🔐 Secure Message Vault

A full-stack MERN application that allows users to securely store and manage private messages using encryption.

Users can register, log in, save encrypted messages, decrypt them when needed, and delete them securely.

---

## 🚀 Features

- 🔑 User Authentication (JWT)
- 🔒 Password Hashing (bcrypt)
- 🔐 AES Encryption for messages
- 📥 Save encrypted messages
- 🔓 Decrypt messages
- ❌ Delete messages
- 🛡 Protected API routes
- ⚛️ React frontend with clean UI

---

## 🛠 Tech Stack

### Frontend
- React.js
- Axios
- CSS (Cybersecurity-themed UI)

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Mongoose)

### Security
- bcrypt (password hashing)
- JSON Web Token (JWT)
- CryptoJS (AES encryption)
